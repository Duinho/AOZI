### ⚙️ 1. 핵심 자동화 스크립트: 인원수 집계 및 보고서 생성

다음 Python 스크립트는 `Master Data Schema`에 맞춰 구조화된 데이터가 입력되었을 때, 요청하신 4가지 조건(학년, 성별, 학교, 요일반)을 기준으로 고유한 학생 수를 집계하고, 그 결과를 `Desktop/Student_Utilization_Report.xlsx` 파일로 저장합니다.

**[필수 전제]**
이 코드가 실행되려면, Researcher가 정의한 스키마를 준수하는 데이터프레임(예: `df_raw`)이 메모리에 로드되어 있어야 합니다.

```python
import pandas as pd

# =====================================================================

def generate_student_report(df: pd.DataFrame):
    """
    학생 ID를 기준으로 학년별, 성별, 학교별, 요일반 분류 인원수를 집계하고 엑셀로 저장합니다.
    """
    print("🔍 데이터 집계 시작: 4차원 인원수 카운트...")

    # 1. 핵심 집계 로직: 지정된 4개 필드를 기준으로 그룹화하고 고유한 학생 ID의 개수를 셉니다.
    aggregation_keys = ['Grade', 'Gender', 'School_Name', 'Day_Type']
    
    # groupby()를 사용하여 모든 조합에 대해 그룹을 만들고, size()로 각 그룹의 크기(인원수)를 계산합니다.
    student_counts = df.groupby(aggregation_keys)['Student_ID'].count().reset_index(name='학생_인원수')

    # 2. 결과 데이터프레임의 컬럼명을 사용자 친화적으로 변경합니다.
    student_counts.columns = [
        '학년', '성별', '학교명', '요일반', '학생_인원수'
    ]
    
    # 3. 최종 보고서 제목 추가 (선택 사항)
    student_counts['보고서_제목'] = '1~3기 학생 활용 현황'
    
    # 4. 엑셀 파일 경로 설정 및 저장
    output_filename = "Student_Utilization_Report.xlsx"
    # 실제 로컬 바탕화면 경로를 지정합니다.
    desktop_path = os.path.join(os.path.expanduser('~'), 'Desktop')
    output_filepath = os.path.join(desktop_path, output_filename)

    try
